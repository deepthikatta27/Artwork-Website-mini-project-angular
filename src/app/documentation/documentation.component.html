Getting Started with Artwork Explorer in Angular

Artwork Explorer is an Angular application that allows users to browse, add, and remove artwork items from a wishlist. This documentation page will explain how to use the application.

Installing Artwork Explorer Source Code

To install the Artwork application source code, follow below mentioned steps:

  1. Clone the repository from GitHub
  2. In a terminal window, navigate to the root directory of the Artwork application:cd artwork
  3. Run the command npm install to install the required dependencies.
  4. Run the command ng serve to start the development server. The application will launch on the localhost.

Prerequisites

To deploy the Artwork application locally on your machine, you'll need the following:

  • Node.js (version 8 or higher)
  • Angular CLI (version 10 or higher)
  • An internet connection

Using Artwork Explorer

The Artwork application has three main features: browsing artwork items, adding items to the wishlist, and removing items from the wishlist.


Browsing Artwork Items

To browse artwork items, simply browse the home page where all available artworks are displayed. To filter the artwork list based on item name and its details, use the search bar in the navigation bar.


Adding Items to the Wishlist

To add an artwork item to the wishlist, simply click the "heart" icon on top-right of item image you want to add. The item will be added to the wishlist, which you can view on the "Wishlist" page of the application and a alert will be displayed indicating that the item is added in the wishlist.


Removing Items from the Wishlist

To remove an artwork item from the wishlist, simply click on the "heart" icon on the image again or navigate to the "Wishlist" section of the application and click the "Remove" button next to the item you want to remove. The item will be removed from the wishlist, alert message will be displayed and will no longer be visible on the page.

Page Details:

  1. Home : This section displays a welcome message and a list of all the artworks available in the museum's collection. Each artwork displayed has a "heart" icon to add the artwork to the wishlist and a "View More" button to display that particular artwork's details.
  2. About : This section provides a brief introduction to the Art Institute of Chicago.
  3. Documentation : This section explains how to use the application, including how to browse, search, and save artworks to the wishlist.
  4. WishList : This section displays a list of all the favourite artworks added to the user's wishlist. Here, each artwork has a "View More" button to navigate to its details view, and a "Remove" button to delete an artwork from the wishlist.
